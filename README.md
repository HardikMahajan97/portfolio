# Portfolio Website

## Overview

This project is a personal portfolio website developed using **HTML, CSS, and JavaScript**. It showcases personal information, technical skills, projects, and contact details through a clean and responsive user interface.

The project was created as part of the **Full Stack Development Lab (FSDL)** practical assignments to demonstrate frontend web development concepts.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

- Responsive design
- About Me section
- Skills showcase
- Projects section
- Contact information
- Smooth navigation
- Interactive user interface
